Mirror brasileiro do npmjs.org

O mirror brasileiro do registro npmjs.org precisa da sua ajuda, estamos procurando por um patrocinador para colocar este servidor em solo brasileiro e ajudar todos os desenvolvedores e empresas brasileiras com uma latência menor. Se você se interessa em ajudar o mirror, envie um email para contato@nodebr.org. Com a sua ajuda pretendemos também comprar um certificado para esse site e operar apenas em cima de conexões criptografadas.

Instalação

Siga as instruções abaixo para começar a utilizar o mirror:

        
  # No seu terminal digite a seguinte linha
  npm config set registry=http://registry.npmjs.com.br

  # Para parar de usar o mirror
  npm config del registry

Como funciona

Este servidor faz cache on demand dos pacotes e dos metadados dos pacotes, por isso a primeira requisição pode demorar um pouco se o seu pacote ainda não estiver em cache no servidor. Os metadados dos pacotes expiram a cada hora, então uma atualização ou nova versão de um pacote pode demorar alguns minutos até que o seu cache de metados expire. Todas as outras requisições são proxiadas diretamente para o https://registry.npmjs.org, assim é possível fazer o login e enviar os seus pacotes diretamente para esse mirror.
